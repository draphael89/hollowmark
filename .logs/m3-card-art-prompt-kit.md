# M3 Card Art Prompt Kit

Date: 2026-04-26

## Scope

- Added production prompts for four card-art targets:
  - Oath Ward
  - Rootfire
  - Shadow Mark
  - Tripwire
- Together with the already-approved Blood Edge card art, these define the first five-card visual production target.

## Boundary

This pass does not generate images, add manifest passports, approve art, or wire new cards into gameplay. The next card-art run should generate candidates from these prompts, process them to 64x48 previews, review them in Visual Gallery, and only then advance manifest state.

## Rationale

The chosen cards cover the core M1 readability families without broadening content:

- debt damage: Blood Edge
- block plus Ward: Oath Ward
- Poison plus debt: Rootfire
- Mark plus debt: Shadow Mark
- Weak plus Bleed: Tripwire
